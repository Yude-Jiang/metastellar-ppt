# Agency Pitch Playbook — client-facing proposals

Default positioning: **a marketing agency presenting a premium solution to a client**.
Tone: confident, minimal, editorial — not a dense internal report.

## Aesthetic principles

| Principle | Do | Avoid |
|-----------|-----|-------|
| **Hierarchy** | One hero idea per slide; oversized titles (32–40pt) | Walls of bullets, 8pt footnotes |
| **Whitespace** | Generous margins; speaker-led pacing | Cramming proof points onto one slide |
| **Credibility** | Real metrics, timelines, named deliverables | Vague adjectives without numbers |
| **Premium feel** | Dark covers, accent rules, 3-up KPI layouts | Clip-art, rainbow palettes, >3 colors |
| **Story arc** | Situation → insight → strategy → plan → proof → ask | Random slide order |

## Recommended deck arc (adapt to page count)

| # | Slide role | Preferred builder |
|---|------------|-------------------|
| 1 | Cover — client + project name | `agency_cover_slide` |
| 2 | Agenda / what we'll cover | `agenda_slide` |
| 3 | Client context / challenge | `challenge_solution_slide` or `big_idea_slide` |
| 4 | Insight / north star | `big_idea_slide` |
| 5 | Strategic pillars (3–4) | `pillar_strategy_slide` |
| 6 | Creative / channel approach | `left_image_icon_rows_slide` or `add_cards_row` |
| 7 | Campaign timeline | `campaign_timeline_slide` |
| 8 | Expected results (KPIs) | `metrics_3up_slide` |
| 9 | Case proof / credentials | `left_image_tiered_list_slide` |
| 10 | Investment / next steps | `bullet_box` + `add_message_bar` |
| 11 | Thank you / Q&A | `closing_slide` |

For **short decks (3–5 slides)**: cover → big idea → pillars or metrics → timeline or challenge/solution → close.

For **reading-first** decks: keep the same arc but allow denser `pillar_strategy_slide` / `add_cards_row`.

## Layout selection by moment

| Client moment | Layout | Why |
|---------------|--------|-----|
| First impression | `agency_cover_slide` | Dark, confident, agency-grade |
| Section change | `agency_section_slide` | Editorial pause, clear structure |
| "The big idea" | `big_idea_slide` | One sentence, maximum impact |
| "Why us / why now" | `challenge_solution_slide` | Classic pitch framing |
| Strategy recommendation | `pillar_strategy_slide` | 3–4 pillars, scannable |
| Proof / ROI | `metrics_3up_slide` | Large numbers, client-ready |
| Rollout plan | `campaign_timeline_slide` | GTM / activation credibility |
| Creative showcase | `left_image_icon_rows_slide` | Hero visual + proof points |

## Copy guidelines

- Headlines: **verb-led or outcome-led** ("Grow qualified pipeline by 40%")
- Message bars: one line, **no more than 12 words**
- Bullets: **fragments**, not sentences; max 5 per block in speaker mode
- Numbers: always pair **value + label + optional benchmark**

## Typography scale (agency pitch)

| Element | Size | Weight |
|---------|------|--------|
| Cover headline | 40pt | Bold |
| Section title | 34pt | Bold |
| Big idea | 36pt | Bold |
| KPI value | 44pt | Bold |
| Slide title | 26–27pt | Bold |
| Message bar | 20pt | Bold |
| Body | 13–14pt | Regular |
