# ST Compliance Checklist

## Quick do / don't

**Do**
- Start from the official ST template (brand.st.com).
- Keep the logo safe zone clear (~2.5× clear space).
- Limit to 2–3 colors; large areas use primary colors.
- Arial only; content 12–20 pt (prefer 14 pt); message bar 20 pt.
- Message bar single text color, fill = yellow / dark blue / light blue / first dark-blue shade.
- "Title Only" layout for content; dedicated layouts for special slides.
- Embed text in shapes; "Do not Autofit"; shade with light grays; yellow for one highlight.
- Lock aspect ratio on images; ST-owned assets; SVG for shapes.
- Use the predefined footer field for footnotes.

**Don't**
- White text on yellow.
- AI-generated visuals (incl. ST ChatGPT). 
- SmartArt.
- Arial Narrow for regular text (OK only inside block-diagram boxes).
- Autofit / Shrink-to-fit on text boxes.
- Graphical elements in the title area (except the sub-brand / longevity stamp).
- More than 2–3 colors per slide.
- Resizing/restyling the message bar beyond the allowed fills and text colors.

## Pre-share housecleaning (do before delivering externally)
1. **Classification** — set the correct document classification. Decks from the Sales &
   Marketing sharing portal (SMS) are **Public**.
2. **Inspect the file** — File → Info → **Check for issues** → **Inspect Document**. Select
   all categories → **Inspect**.
3. **Remove metadata at this stage** (Document Properties & Personal Information, revision
   tracking, cropped-out image data, etc.). You will add fresh metadata back afterward.
4. **Keep relevant presentation notes; remove internal notes.**
5. **Add new metadata** after cleaning (File → Properties: Title, Tags, etc.).
6. **Review** in **Slide Sorter** (flow/consistency) and **Slide Show** (final impression,
   image/icon/animation behavior, screen/display differences). Icons are in the
   bottom-right of the editing window.

## Mandatory closing / trademark slide
End external decks with the ST closing slide. Footer text must read:

```
© STMicroelectronics - All rights reserved.
ST logo is a trademark or a registered trademark of STMicroelectronics International NV
or its affiliates in the EU and/or other countries.
For additional information about ST trademarks, please refer to www.st.com/trademarks.
All other product or service names are the property of their respective owners.
```

The canonical closing visual is the ST Dark Blue "Our technology starts with You" slide with
the yellow trademark footer band and the ST logo.

## Useful ST resources
- **brand.st.com** — guidelines, latest template, slide-layout library, assets (prefer SVG).
- **Sales & Marketing sharing (SMS)** — 300+ reusable, public-approved presentations
  (products, applications, ecosystem, company), refreshed quarterly.
